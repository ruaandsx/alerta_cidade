# Frontend - AlertaCidade

## Setup Rápido

```bash
npm install
npx expo start         # Escaneie QR code com Expo Go
```

## Estrutura

- `src/screens/` — Telas (Login, Feed, Mapa, etc)
- `src/components/` — Componentes reutilizáveis
- `src/services/` — Requisições à API
- `src/store/` — Contexto de autenticação
- `src/navigation/` — Navegação entre telas

## O que codar

1. `screens/NewReport` — Formulário de denúncia
2. `screens/ReportDetail` — Detalhes + comentários
3. `screens/Map` — Mapa com pins
4. `screens/Profile` — Perfil do usuário
5. `components/` — Componentes reutilizáveis
